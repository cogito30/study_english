# 📅 일본어 주제별 학습 로드맵

2. 일상생활과 시간 (기상, 수면, 하루 일과, 요일, 날짜)
3. 음식과 식당 (메뉴 읽기, 맛 표현, 식당에서 주문하기)
4. 취미와 여가 생활 (영화, 음악, 스포츠, 관심사 묻고 답하기)
5. 위치와 길 찾기 (방향, 장소, 대중교통 이용하기)

8. 건강과 신체 (신체 부위, 증상 설명, 병원/약국 방문)
9. 날씨와 계절 (계절별 특징, 일기예보, 자연)
10. 감정과 성격 (기분 표현, 성격 묘사, 칭찬 및 위로)
11. 직장과 학교 (업무, 전공, 학교생활, 면접 기본 표현)
