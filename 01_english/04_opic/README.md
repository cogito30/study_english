# [OPIc](https://www.opic.or.kr/opics/jsp/view/index.jsp)

## Intro
- 시험 비용: 84,000원(2026.07 기준)
- 응시 조건: 25일 내에 응시 기록이 없는 사람
- 시험 방식: iBT 기반의 외국어 말하기 평가
- 시험 목적: 실제 생활에서 얼마나 효과적이고 적절하게 해당 언어를 사용할 수 있는가를 측정
- 평가 방식: NL < NM < NL < IL < IM < IH < AL
- 시험 시간: 총 60분(오리엔테이션: 20분 + 시험: 40분)
- 문항 수: 총 12-15문항(선택형 주제 2세트 + 공통형 주제 2세트 + 롤플레이 1세트)

#### 시험 진행 순서

(오리엔테이션: 20분)
1. 사전 설문 조사(Self-Assessment)
- 평가 문항을 위한 사전 설문 진행
2. 자기평가
- 시험의 난이도 결정
3. 사전 점검
- 질문 청취 및 답변 녹음 기능을 사전 점검
4. 샘플 문제 답변
- 시험 진행 방법 안내

(본 시험: 40분)
1. 1st Session
- 선택한 난이도를 바탕으로 7개의 문제 진행
2. 난이도 재조정
- 2차 난이도 설정
3. 2nd Session
- 나머지 인터뷰 약 7문제 출제

#### Self-Assessment
- 1단계: 10단어 이하의 단어로 말하기
- 2단계: 기본적인 단어 말하기
- 3단계: 기본 정보를 간단한 문장으로 말하기
- 4단계: 일상에 대해 간단한 대화 가능
- 5단계: 사회적 관심사에 대해 대화 가능
- 6단계: 다양한고 어려운 주제에 대해 다양한 어휘를 사용하여 대화 가능
=> 등급에 따른 추천: IM(3단계), IH(4단계), AL(5단계)
