# Day18: 사내 IT 및 전산 문제
- 사내 IT 및 전산 문제 (IT & Technical Issues in the Workplace)

## 명사
1. network: 네트워크, 통신망
- The company network is currently down for maintenance.
- 회사 네트워크가 현재 점검을 위해 다운되었습니다(중단되었습니다).

2. server: 서버
- We couldn't save the files because the main server crashed.
- 메인 서버가 다운되어서 우리는 파일들을 저장할 수 없었습니다.

3. access: 접근, 접속 권한
- I need to request access to the new database.
- 저는 새 데이터베이스에 대한 접속 권한을 요청해야 합니다.

4. update: 업데이트, 갱신
- Please make sure to install the latest software update.
- 반드시 최신 소프트웨어 업데이트를 설치하시기 바랍니다.

5. backup: 백업, 예비 저장
- I always keep a backup of my important files on a USB drive.
- 저는 항상 중요한 파일들의 백업을 USB 드라이브에 보관합니다.

6. error: 오류, 에러
- An unknown error occurred while printing the document.
- 문서를 인쇄하는 동안 알 수 없는 오류가 발생했습니다.

7. password: 비밀번호
- You are required to change your password every three months.
- 당신은 3개월마다 비밀번호를 변경해야 합니다.

8. virus: 바이러스, 악성 코드
- My computer was infected with a dangerous computer virus.
- 제 컴퓨터가 위험한 컴퓨터 바이러스에 감염되었습니다.

9. system: 시스템, 전산 체계
- The security system will be upgraded next week.
- 보안 시스템이 다음 주에 업그레이드될 것입니다.

- If you have any computer issues, please call tech support.
- 컴퓨터에 문제가 생기면 기술 지원팀에 전화하세요.

## 동사
1. install: 설치하다
- The IT guy came over to install a new antivirus program.
- 전산팀 직원이 와서 새로운 백신 프로그램을 설치해 주었습니다.

2. crash: 다운되다, 갑자기 멈추다
- I lost all my work because my laptop suddenly crashed.
- 노트북이 갑자기 다운되는 바람에 작업하던 것을 모두 날렸습니다.

- The screen is frozen. You should reboot the computer.
- 화면이 멈췄어요. 컴퓨터를 재부팅해야 합니다.

4. troubleshoot: 문제를 해결하다, 원인을 규명하다
- The technician is trying to troubleshoot the network problem.
- 기술자가 네트워크 문제의 원인을 규명하여 해결하려고 노력 중입니다.

5. log in: 로그인하다, 접속하다
- I can't log in to my email account because I forgot my password.
- 비밀번호를 잊어버려서 이메일 계정에 로그인할 수 없습니다.
